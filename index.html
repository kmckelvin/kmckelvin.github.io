Hi! My name's Kevin. I'm an autodidact software developer, speaker and blogger. I'm CTO with the talented team at Resource Guru where we build the best team scheduling tool on the web :). I currently live in Newcastle upon Tyne, UK, having previously worked as a Lead Ruby and JavaScript developer at Platform45 in Johannesburg, South Africa.

I prefer working with open-source software. I use VS Code on macOS, although you'll have to crowbar my vim bindings from my cold, dead hands! My preferred hammers are TypeScript, Go and Ruby.

I've been writing software for as long as I can remember, first on my Dad's Commodore 64 before learning QBASIC on DOS, followed by Pascal, Delphi, C and PHP during primary and high school years. My professional career started off in the .NET stack, writing C# with .NET 3.5 and 4.0, as well as working on data intensive OLTP and OLAP systems using Microsoft tooling such as SQL Server and Analysis Services. I progressed into Ruby and JavaScript in 2011 with Ruby on Rails. Since then I've spent countless hours honing my skills on the journey to mastering the art of software development.

I believe a good craftsman should be willing to teach and mentor others. I regularly attend and speak at usergroup events and conferences. I was a regular panelist on the ZADevChat podcast from 2015-2017 with Kenneth Kalmer, Len Weincier and Chantal Yang.
